## 功能规划
```
blogo init 初始化blog
blogo add  <uri> [title] 新建一个博客
blogo publish 发布


blog server 内置一个server服务预览


1 文件创建
2 发布

3 模板机制
4 插件机制
5 内部编辑器
6 独立处理单个件(md => html)
```

## 项目目录
```
_posts/         存放md文件
output/         生成的html文件
templates/      模板文件
    default     默认的模板
widgets/        挂件存放目录
blogo.yml       项目配置文件
```
